# GHUMAKKAD

> Explore More. Pay Less.

## About

GHUMAKKAD is an AI-powered travel and adventure marketplace where users can compare prices from multiple verified vendors and book experiences at the best available price.

## Features

- Adventure Booking
- Price Comparison
- Verified Vendors
- AI Trip Planner
- Secure Payments
- Vendor Portal
- Admin Dashboard

## Tech Stack

- Flutter
- Firebase
- Gemini AI
- Google Maps
- Razorpay
